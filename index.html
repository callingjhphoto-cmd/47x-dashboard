API Reference

47X Data Engine REST API — https://47x-audit-platform.truegroup.workers.dev/ — All responses use the standard ApiResponse envelope.

Response Envelope

Every endpoint returns this structure

{
  "success": boolean,
  "data": T | null,
  "error": string | null,
  "meta": {
    "companyId": string,
    "service": ServiceName | "all",
    "collectedAt": string,     // ISO 8601
    "confidence": number,      // 0.0 - 1.0
    "dataPointCount": number,
    "stale": boolean           // true if data past expiry
  }
}
